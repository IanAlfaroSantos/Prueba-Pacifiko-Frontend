# Catalog Frontend

Frontend en React + Vite para el catálogo de productos y carrito.

## Ejecutar

```bash
npm install
npm run dev
```

Si desea conectar el frontend al backend local cambiar en api.js:

```bash
http://localhost:3000/catalog/v1/
```

Puedes cambiar la URL base en:

```bash
src/services/api.jsx
```
